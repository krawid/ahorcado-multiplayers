# Carpeta de Sonidos

## Instrucciones

Coloca aquí el archivo de sonido de aplausos con el nombre:
- `applause.mp3` (formato recomendado)

O alternativamente:
- `applause.wav`
- `applause.ogg`

El sistema intentará reproducir este archivo cuando el jugador gane. Si el archivo no existe o falla la reproducción, se usará automáticamente un sonido sintetizado como respaldo.

## Formatos soportados

Los navegadores modernos soportan:
- MP3 (mejor compatibilidad)
- WAV (sin compresión, archivos más grandes)
- OGG (buena compresión, buena calidad)
